# Frontend — Performance

1. Lazy load all pages with React.lazy + Suspense
2. Memoize expensive computations with useMemo
3. Use useCallback for event handlers passed to child components
4. Zustand selectors should be atomic (select only what you need)
5. Avoid unnecessary re-renders with proper component splitting
6. Use Framer Motion's layout animations sparingly
7. Images (when added) should be optimized
