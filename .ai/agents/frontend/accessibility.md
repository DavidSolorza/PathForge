# Frontend — Accessibility

1. All buttons must be keyboard-accessible (native button or role="button" + tabIndex)
2. Forms: labels associated with inputs, error messages announced
3. Color contrast: minimum 4.5:1 for text
4. Interactive elements have visible focus rings (ring-2 ring-primary-500)
5. Loading states use aria-busy or role="status"
6. Images (when added) have alt text
7. Meaningful heading hierarchy (h1 -> h2 -> h3)
