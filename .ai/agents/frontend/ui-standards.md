# Frontend Rules

1. Use TypeScript strict mode
2. Use TailwindCSS v4 for all styling (no CSS modules or styled-components)
3. Every page must handle: loading, empty, error, and success states
4. Lazy load all route pages
5. Use Zustand selectors to prevent unnecessary re-renders
6. No inline styles
7. No emojis in any UI text
8. Icons from Lucide React only
