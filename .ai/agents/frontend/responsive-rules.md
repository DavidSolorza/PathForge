# Frontend — Responsive Rules

1. Mobile-first: base styles are mobile, then md: lg: xl: overrides
2. Breakpoints: sm: 640px, md: 768px, lg: 1024px, xl: 1280px
3. Grid layouts: grid-cols-1 md:grid-cols-2 lg:grid-cols-3
4. Sidebar: hidden on mobile, overlay with backdrop on toggle
5. Tables: horizontal scroll on mobile
6. Forms: full-width on mobile, max-w-md on desktop
7. Font sizes: text-sm on mobile, text-base on desktop where appropriate
