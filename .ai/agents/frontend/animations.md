# Frontend — Animations

- Framer Motion for all animations
- Page transitions: fade + slight slide up (initial: opacity 0, y: 20)
- List items: stagger children with delay based on index
- Hover effects: scale 1.02-1.05, smooth transition 200ms
- Loading: bounce animation for dots, spin for Loader2 icon
- Sidebar: slide transition 300ms
- Toast: slide in from right, fade out
- Keep animations subtle — never flashy
