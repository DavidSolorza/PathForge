# Documentation — Onboarding

1. Clone repo: `git clone <url> && cd pathforge-ai`
2. Install: `pnpm install`
3. Set up .env: copy .env.example to .env, fill in values
4. Run dev: `pnpm dev` (frontend), `cd server && pnpm dev` (backend)
5. Run tests: `pnpm test`
6. Build: `pnpm build`
7. Demo user: demo@pathforge.ai / 123456
