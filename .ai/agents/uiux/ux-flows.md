# UI/UX — UX Flows

1. **New user**: Landing -> Register -> Create first path -> See path stages -> Complete topics
2. **Returning user**: Dashboard (stats + active paths) -> Continue learning -> Complete topics -> Ask AI
3. **Path creation**: AIAssistantPage -> Type goal -> Survey (time/method/level) -> Generate -> Redirect to path
4. **Topic completion**: Click checkbox -> Mark as done -> Update stats -> Update activity feed
5. **Ask AI**: Type question -> AI analyzes progress -> Recommendation with reasoning
