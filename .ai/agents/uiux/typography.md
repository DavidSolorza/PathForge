# UI/UX — Typography

- System font stack (default Tailwind): Inter, system-ui, sans-serif
- Monospace for code: 'Fira Code', 'JetBrains Mono', monospace
- Headings: font-semibold
- Body: font-normal
- Code: font-mono text-sm bg-neutral-100 rounded px-1
- Line height: leading-relaxed for body text
- No font sizes below 12px (text-xs minimum)
