# UI/UX — Hierarchy

- Page title: text-2xl font-semibold
- Section title: text-lg font-semibold
- Card title: text-base font-semibold (or text-sm for compact)
- Body: text-sm text-neutral-700
- Caption: text-xs text-neutral-500
- Labels: text-xs font-medium uppercase tracking-wider
- Primary actions: solid button (bg-primary-600)
- Secondary actions: outline or ghost button
- Disabled states: opacity-50 cursor-not-allowed
