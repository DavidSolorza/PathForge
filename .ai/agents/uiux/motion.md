# UI/UX — Motion

- Page transitions: fade + slide up (opacity: 0, y: 20 -> opacity: 1, y: 0) — 400ms
- Card hover: subtle shadow increase + slight translateY(-1px) — 200ms
- Button hover: bg shade change — 150ms
- Button active: scale(0.97) — 100ms
- Sidebar: slide 300ms ease
- Toast: slide in from right, auto-dismiss after 4s
- Modal: backdrop fade + content scale, 200ms
- Loading dots: bounce animation, staggered 150ms
- Reduce motion: respect prefers-reduced-motion
