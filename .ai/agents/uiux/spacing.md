# UI/UX — Spacing

- Page padding: px-4 py-8 lg:px-8
- Card padding: p-4 (16px)
- Section gap: space-y-8 (32px)
- Stack gap: space-y-6 (24px)
- Element gap: gap-3 (12px) or gap-4 (16px)
- Icon spacing: gap-2 (8px) between icon and text
- Consistent: use Tailwind's spacing scale exclusively
- Never use arbitrary values like px-[13px]
