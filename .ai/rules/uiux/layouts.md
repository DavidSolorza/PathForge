# UIUX Layout Rules

1. Main content: mx-auto max-w-7xl
2. Sidebar: w-64, collapsible to w-16
3. Cards: rounded-xl, border border-neutral-200, bg-white
4. Card grid: grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4
5. Forms: max-w-md for centered forms, full width for inline
6. Modal: fixed inset-0, centered, with backdrop blur
