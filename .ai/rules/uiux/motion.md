# UIUX Motion Rules

1. Use Framer Motion for all animations
2. Duration: 0.2s for micro-interactions, 0.3s for page transitions
3. Easing: default Framer Motion ease
4. Stagger children: staggerChildren with 0.05s delay
5. Page load: fade in + slide up (initial: opacity 0, y: 20)
6. Only animate when it serves a purpose (feedback, focus, navigation)
7. No gratuitous animations
