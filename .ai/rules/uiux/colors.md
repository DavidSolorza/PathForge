# UIUX Color Rules

1. Primary: blue-600 / blue-50 background
2. Success: green-500 / green-50
3. Warning: amber-500 / amber-50
4. Error: red-500 / red-50
5. Neutral: neutral-900 (text), neutral-500 (secondary), neutral-200 (borders), neutral-50 (bg)
6. Dark mode: dark:bg-neutral-900, dark:text-neutral-100, etc.
7. No emojis: use initial-letter boxes or text badges instead
