# Optimize Performance Prompt

Find and fix performance issues:

1. Check for unnecessary re-renders in components
2. Review Zustand store subscriptions (use selectors)
3. Check bundle size (lazy loading, code splitting)
4. Review API calls (caching, debouncing)
5. Check memory leaks (useEffect cleanup)
6. Run build and verify

Focus: [RENDERING | BUNDLE | API | MEMORY]
