# Create API Prompt

1. Design the endpoint following REST conventions
2. Check if Mongoose model exists or needs creation
3. Create route handler with validation and auth
4. Add error handling
5. Register the route in server/src/index.ts
6. Update frontend ApiAdapter if needed
7. Run server typecheck
