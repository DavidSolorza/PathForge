# Project Goals

## Current Phase
- Polish the existing learning path experience
- Migrate from localStorage to MongoDB for persistence
- Improve AI survey/pre-path questionnaire
- Make topic content display rich and useful

## Short-term Goals
- Full MongoDB integration via DbAdapter pattern
- Real-time sync across devices
- Proper JWT authentication with the backend
- Docker containerization
- CI/CD pipeline

## Medium-term Goals
- Multi-language support (code examples in multiple languages)
- Community learning paths (share/import)
- Progress analytics dashboard
- Spaced repetition reminders

## Long-term Vision
- AI mentor that adapts to learning style in real-time
- Peer learning groups
- Integration with code repositories (GitHub)
- Skill verification through coding challenges
- Marketplace for expert-created paths

## Non-Goals
- Social media features
- Video hosting
- Non-tech topics
