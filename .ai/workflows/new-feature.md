# New Feature Workflow

1. **Planner**: Analyze request, break into tasks, identify risks
2. **Architect**: Design structure, modules, boundaries
3. **Database**: Design/update schemas
4. **Backend**: Create/update APIs
5. **Frontend**: Create/update interfaces
6. **UIUX**: Refine visual experience
7. **Reviewer**: Validate quality, security, performance
8. **Testing**: Add/update tests
9. **DevOps**: Update deployment if needed
10. **Documentation**: Update docs
